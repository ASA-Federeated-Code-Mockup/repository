Summary:

PsyToolkit is a set of software tools for programming psychological experiments on Linux computers. You can design and run your own experiments or questionnaire surveys online (browser-based). You can find and use lots of ready-made experiments or questionnaire on this website. It is suitable for teaching psychology, starting from secondary education. Ideal for student and professional research projects.
